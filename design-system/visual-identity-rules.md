# Visual Identity Rules

## Core Style

This portfolio system uses a clean healthcare operations identity that is structured, professional, and easy to scan.

## Required Design Rules

- Use a pure white background: #FFFFFF
- Use Tennessee Orange only as the accent color: #FF8200
- Use black for all primary text: #000000
- Keep layouts clean and structured
- Use strong spacing and clear sections
- Use tables, workflow maps, and proof statements
- Avoid decorative clutter
- Avoid extra colors

## Visual Elements

Use these elements consistently:

- Tennessee Orange top bars
- Tennessee Orange divider lines
- Tennessee Orange workflow markers
- Black section titles
- Black body text
- White background space
- Simple tables
- Clean proof-of-work structure

## Brand Voice

The visual identity should communicate:

- operational clarity
- healthcare workflow awareness
- denial prevention thinking
- patient access understanding
- revenue cycle discipline
- professional proof-of-work
