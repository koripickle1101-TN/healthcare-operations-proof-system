# Brand Colors

This repository uses a strict three-color brand system.

| Brand Role | Hex Code | Use |
|---|---:|---|
| Primary Background | #FFFFFF | Clean backgrounds, portfolio pages, case studies, documentation space, and visual layouts |
| Accent Color | #FF8200 | Tennessee Orange accent bars, dividers, workflow markers, urgency points, and proof highlights |
| Font Color | #000000 | Headings, body copy, table labels, workflow steps, and proof statements |

## Brand Rule

Use only white, Tennessee Orange, and black.

Do not add blue, green, red, purple, gradients, or extra accent colors.

## Visual Purpose

The brand system is built around white space, black clarity, and Tennessee Orange urgency. The goal is to make healthcare operations work look clean, focused, professional, and recruiter-readable.
